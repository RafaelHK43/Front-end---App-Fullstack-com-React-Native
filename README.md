# 📱 Biblioteca Digital - Mobile App (React Native + Expo)

Este é o aplicativo mobile do sistema de Biblioteca Digital. Ele foi desenvolvido utilizando **React Native com Expo** e consome a API do backend para realizar um CRUD completo de livros por meio de uma interface moderna, limpa e responsiva.

## 🛠️ Tecnologias Utilizadas

* **React Native** (Framework para apps nativos multiplataforma)
* **Expo (v54.0.8)** (Plataforma e ecossistema de desenvolvimento mobile)
* **Axios** (Cliente HTTP para consumo de APIs)
* **JavaScript (ES6+)**
